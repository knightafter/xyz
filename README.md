
# Interview-Oriented Software Engineering Study Plan

This repository contains a **clear, structured study plan** designed to prepare for **Android, Backend, and Full‑Stack software engineering interviews**, with a strong focus on fundamentals and future‑proof skills.

## Tracks Covered

### 1. Android Development (Kotlin)
- Kotlin language fundamentals
- Android lifecycle & core concepts
- MVVM architecture & Jetpack components
- Dependency Injection (Dagger / Hilt)
- Networking, performance, and Jetpack Compose

### 2. Backend Development (Python)
- Python fundamentals
- REST APIs & HTTP concepts
- FastAPI framework essentials
- Backend architecture & authentication

### 3. Databases & SQL
- Relational database fundamentals
- SQL queries (JOINs, GROUP BY, subqueries)
- Indexes, normalization, transactions
- Firebase vs SQL comparison

### 4. Data Structures & CS Core
- Arrays, hash maps, stacks, queues
- Linked lists & trees
- Big-O notation

### 5. AI & Vibe Coding
- Using AI as a productivity tool
- Reviewing and fixing AI-generated code
- System thinking and problem solving

## Purpose
This plan is **interview-focused**, **startup-ready**, and **aligned with modern hiring expectations**, especially for markets like Saudi Arabia.
