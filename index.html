Interview-Oriented Study Plan

Android • Backend • Databases • CS Fundamentals

Track 1: Android Development (Kotlin)

  • Kotlin Basics: val/var, data classes, sealed classes, null safety, lambdas, coroutines
  • Android Core: Activity & Fragment lifecycle, intents, context, configuration changes
  • Architecture: MVC, MVP, MVVM, ViewModel, Repository pattern
  • Jetpack: ViewModel, LiveData, StateFlow, Room, Navigation
  • Dependency Injection: Dagger, Hilt, scopes, @Inject, @Module
  • Networking: REST APIs, Retrofit, OkHttp, error handling
  • Jetpack Compose: Composables, state, recomposition, side effects
  • Performance: Memory leaks, ANR, caching, pagination

Track 2: Backend Development (Python)

  • Python Basics: data types, functions, classes, async vs sync
  • Backend Concepts: APIs, REST, HTTP methods, status codes, JWT
  • FastAPI: routes, Pydantic models, dependency injection, auth
  • Architecture: layered architecture, services, repositories

Track 3: Databases & SQL

  • DB Fundamentals: relational vs NoSQL, tables, schema, ACID
  • SQL Core: SELECT, WHERE, JOIN, GROUP BY, subqueries
  • Advanced Concepts: indexes, normalization, transactions
  • Firebase vs SQL: use-cases and tradeoffs

Track 4: Data Structures & CS Core

  • Arrays, strings, hash maps
  • Stacks, queues, linked lists
  • Trees (basics)
  • Big-O notation

Track 5: AI & Vibe Coding

  • Prompting AI for code
  • Reviewing and refactoring AI output
  • Using AI for testing and documentation
  • Designing systems, not just writing code

Designed for interview preparation & future-ready engineers
